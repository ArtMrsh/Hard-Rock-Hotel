Для начала работы сайта, нужно установить все необходимые модули, выполнив комманду "npm install", находясь в корневой папке. Далее нужно запустить сервер, ипользуя команду "nodemon index.js" в папке server. После чего выполнить комманду "gulp", находясь в корневой папке проэкта. P.S. gulpfile.js настроен не до конца, галерея просто для  видимости, своя уже в процессе. Всё это будет исправлено до защиты.
